# TAM Proxies

## SaladCloud
Distributed consumer GPU cloud network. Gamers install an app, earn gift cards and rewards while their PC sits idle. Customers get AI inference at up to 90% cheaper than AWS.
- 1 million registered node operators
- 60,000+ daily active GPUs
- 191 countries
- $27.5M raised, 11-50 person team

## Vast.ai
Peer-to-peer GPU marketplace where individuals rent out idle hardware to AI developers and researchers.
- 17,000 GPUs
- 1,300 independent providers
- 123,000 customers on demand side

## Bitcoin Nodes
Individuals running full nodes to independently verify transactions. Ideologically motivated, technically demanding, no financial reward.
- ~24,500 reachable nodes globally
- 15+ years of organic growth

## Folding@home
Distributed computing project for protein folding research (Alzheimer's, cancer, COVID-19). No financial incentive — pure altruism.
- ~30,000 daily active contributors pre-COVID (after 20 years)
- 1 million+ contributors during COVID surge in one month
- Zero financial incentive — grew entirely on narrative

## SETI@home
Distributed computing project searching for extraterrestrial intelligence using idle home computers.
- Peak ~5 million contributors
- Free to join, screensaver model, pure altruism

## Airbnb Hosts
Individuals monetizing idle physical property (homes, rooms) through a centralized marketplace platform.
- 5 million hosts worldwide
- Average US host earns $14,000/year
- Requires owning property in a desirable location — high barrier vs compute nodes

## Netflix Subscribers
Consumer subscription service — proxy for mainstream non-technical mass adoption.
- ~300 million subscribers globally
- Represents the ceiling of what consumer network adoption looks like at scale

## Summary Framework

| Proxy | Scale | Incentive | Barrier |
|---|---|---|---|
| SaladCloud | 1M nodes / 60K daily active | Gift cards, rewards | Low — install app |
| Vast.ai | 1,300 providers | Cash | Medium — need GPU setup |
| Bitcoin nodes | 24,500 | None (ideological) | Medium — technical |
| Folding@home | 30K steady / 1M COVID peak | None | Very low — screensaver |
| SETI@home | 5M peak | None | Very low — screensaver |
| Airbnb | 5M hosts | $14K/year | High — must own property |
| Netflix | 300M subscribers | N/A (demand side) | Very low — pay monthly |

## Scenario Planning

| Scenario | Node Count | Rationale |
|---|---|---|
| Bear | 24,500 | Bitcoin node territory — stays niche and technical |
| Base | 500,000 | Folding@home / Airbnb territory — financial incentive working, word of mouth spreading |
| Bull | 5 million | Airbnb scale — economics compelling enough for mainstream tech-adjacent people |
| Moonshot | 100 million | Netflix territory — requires one-click setup and meaningful passive income for non-technical users |

**Key insight:** SaladCloud proved 1 million nodes is achievable with $27.5M raised and a small team — purely by nailing supply-side incentives (gamers + gift cards) and minimizing setup friction (install app, earn passively). The constraint is not hardware availability — it is onboarding friction and incentive clarity.
